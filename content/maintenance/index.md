## 🎧 iDream5S Maintenance

{{< box "🧽 Cleaning Guide" >}}
### Surface Cleaning
```
✓ Use soft, damp cloth with mild soap
✓ Wipe entire exterior gently
✗ Avoid getting water inside device
✓ Dry thoroughly before storage
```

### Eye Mask Cleaning
```
1. Remove eye mask (if detachable)
2. Hand wash with mild detergent
3. Rinse thoroughly
4. Air dry completely
5. Reattach when fully dry
```

### Massage Heads
```
✓ Use cotton swab between heads
✓ Remove hair/debris buildup
✗ No harsh chemicals
✓ Check for wear monthly
```

{{< box "🔋 Battery Care" >}}
### Best Practices
| Do | Don't |
|----|-------|
| ✅ Charge when battery is low | ❌ Overcharge (unplug when full) |
| ✅ Store at 50% for long-term | ❌ Store fully depleted |
| ✅ Use original charger | ❌ Use third-party chargers |
| ✅ Cool, dry storage | ❌ Humid environments |

**Battery Life:** Up to 1.5 hours (4-6 sessions) per charge

## 🛠️ Troubleshooting Guide

{{< box "🚨 Device Won't Turn On" >}}
### Step-by-Step Diagnosis

1. **Check Battery**
   ```
   ✓ Is the charging light on when plugged in?
   ✓ Try different outlet
   ✓ Check cable connection
   ```

2. **Check Power Button**
   ```
   ✓ Press and hold 3-5 seconds
   ✓ Try multiple times
   ✓ Check for physical damage
   ```

3. **Inspect for Damage**
   ```
   ✓ Look for cracks or dents
   ✓ Check charging port
   ✓ Inspect power button
   ```

4. **Reset Device**
   ```
   ✓ Refer to user manual
   ✓ Follow reset procedure
   ✓ Try powering on again
   ```

5. **Contact Support**
   ```
   ✓ If none of above works
   ✓ Have purchase receipt ready
   ✓ Note device serial number
   ```
{{< /box >}}

   ```

   ✓ Dry thoroughly
   ```

3. **Storage Environment**
   ```
   ✓ Cool temperature
   ✓ Low humidity
   ✓ No direct sunlight
   ✓ Stable surface
   ```
